# ubuntu-server-lab

Laboratorio de Infraestructura

En este repositorio se documentara la instalacion y configuracion de un servidor Ubuntu Serverm, utilizado como un laboratorio para aprender admministracion de sistemas y los fundamentos de pentest de infraestructura

Objetivos:
- Aprender a instalar Ubuntu Server
- Aprender la administracion basica de Usuarios y permisos
- Simular un entorno empresarial simple

Alcance del proyecto:
-Instalacion de Ubuntu Server en una maquina virtual
-Realizar una configuracion basica del sistema
-Gestionar usuarios y permisos

Entorno:
-Hipervisor: VirtualBox
-Sistema Operativo: Ubuntu Server
-Red: Laboratorio local
