Reforzamiento ssh en servidor

Reforzar un servidor se entiende por tomar medidas para proteger las conexiones realizadas contra ataques de fuerza bruta, esto usando una llave de autenticacion unica o deshabilitando el login de root, por mencionar algunas

En esta carpeta, se mostraran algunas medidas que se han tomado para reforzar el servidor de manera mas optima y mantener la seguridad del sevidor, para despues realizar pruebas de penetracion y tomar las medidas necesarias, con la finalidad de acostumbrase a un entorno de trabajo mas realista y mantener la seguridad del servidor
